Neural Network and Deep Learning Basics¶

Architecture:¶

  • Input layer
  • Hanya satu layer
  • Jumlah nodes sesuai dengan dimensi variabel prediktor
  • Hidden Layer
  • Jumlah layer bebas
  • Output Layer
  • Hanya satu layer
  • Jumlah nodes sesuai dengan dimensi target
  • Apabila dilakukan one hot encoding, maka jumlah nodes sesuai dengan jumlah kelas
  • Apabila tidak di dilakukan one hot encoding, maka jumlah nodes pasti 1

Learning Process¶

  • Neural Network belajar secara bertahap dan iteratif (berulang-ulang)
  • 1 iterasi = 1 step = 1 epochs = 1 feedforward + 1 backpropagation
  • Feedforward:
  • Proses model memberikan (feed) input kepada semua layer sampai didapatkan suatu prediksi
  • Tahapan yang ada pada proses feedforward:
  1. Input memasukkan data pada layer selanjutnya
  2. Muncul bobot yang dipilih secara random
  3. Dihasilkan prediksi (y')
  4. Prediksi akan dihitung selisihnya dengan target oleh loss_function
  5. Didapatkan error
  • Backpropagation:
  • Proses model menilai error dan kembali (back) untuk mengupdate bobotnya
  • Tahapan yang ada pada proses backpropagation:
  1. Error didapat akan disesuaikan oleh optimizer
  2. Nilai bobot terupdate sebagai sinyal feedback
  • Seberapa cepat model belajar tergantung kecepatan (learning rate)

Panduan Model Building dan Tuning¶

  • Nodes pada input layer: sejumlah prediktor yang kita miliki
  • Nodes pada output layer:
  • regresi = 1
  • klasifikasi = sejumlah kelas variabel target
  • Hidden layer
  • Jumlah hidden layer (how deep):
  • Model lebih kompleks
  • Waktu training lama
  • Percobaan pertama bisa 2 layer terlebih dahulu
  • Jumlah node pada hidden layer (how wide)
  • Terlalu banyak bisa menghasilkan overfitting
  • bisa pakai angka kelipatan 2^n (2,4,8,16,32,64, dst...), atau sekita 2/3 jumlah prediktor
  • Jumlah mengerucut semakin ke output layer
  • Loss Function:
  • Regresi: mean squared error (mse)
  • Klasifikasi: Cross-entropy (ce)
  • Activation function
  • Ouput Layer
  • Linear (default keras)
  • Range -inf ~ + inf
  • Untuk regresi
  • Sigmoid
  • range 0~1
  • cocok untuk kasus binari klasifikasi
  • Softmax
  • range 0~1
  • cocok untuk multiclas
  • Hidden Layer
  • ReLu
  • range 0 ~ inf
  • cocok untuk data gambar / image
  • sering digunakan pada hidden layer
  • menghilangkan efek negatif pada data
  • tanh
  • range -1 ~ 1
  • cocok untuk prediktor yang banyak negatifnya
  • Metode back propagation (optimizer)
  • Gradient descent
  • Adam
  • Learning rate
  • Batch yang banyak (batch_size = kecil) untuk mencegah komputasi terlalu besar dalam 1 waktu
  • Epoch yang semakin banyak maka:
  • Poses training semakin lama
  • Error lebih kecil
  • Rentan untuk overfitting
  • Dapat ditentukan epoch yang tidak terlalu tinggi dulu seperti 10 atau 15
